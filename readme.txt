Inserts cross linking into your site from post names

== Description ==

This plugin makes any phrases in a post that match the title of another post into a link to that post.


== Installation ==

1. Upload `ES_Cross_Linker.zip` using the Plugin Uploader in the Wordpress admin pages
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit settings on the by going to Settings> Tag Highlighter
